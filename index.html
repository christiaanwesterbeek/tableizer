FAQ

What is TABLEizer?

A tool that converts tabular data into formatted HTML or fixed width text columns.

Do you log my data?

No, this is a client side tool. All is done with Javascript. None of your tabular data is sent to the server.

Who made this?

Christiaan Westerbeek made it. Lots was lended.

  • Table css for the formatted output by Danny Sanchez (http://tableizer.journalistopia.com/).
  • Draggable columns by Romulo do Nascimento Ferreira (http://bytes.com/topic/javascript/insights/750692-drag-drop-table-columns-new-version-explained).
  • Page layout from jsonlint.org.
  • Transposition thanks to a oneliner of lodash (https://lodash.com/docs#zip).
  • And last but not least Angular for not having to fiddle with the DOM myself.
  • The glue by me.
